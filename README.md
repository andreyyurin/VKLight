# VKLight

## Description 

This app Easy and Light Client for VK, special for people, who need to check only news feed and messaging.

## Libs

This app features the use of the following libs:

- Dagger 2
- Coroutines
- Moxy
- Cicerone
- VK Sdk
- Fresco
